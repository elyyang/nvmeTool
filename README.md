# NVMe Test Tool 2.0 #
-------------------------------------------------
## Getting started

### bind uio driver to nvme device

  $ sudo ./scripts/setup.sh 
  or
  $ sudo ./scripts/setup.sh config

### setup udma driver

  $ sudo ./scripts/setup_udma.sh 

### build

  $ sudo make

### run

  $ sudo ./build/nvmeTool

### reset

  $ sudo ./scripts/setup.sh reset
